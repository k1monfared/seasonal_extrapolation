Coverage shown on each estimate is the Monte Carlo probability that the interval contains the true impact, across many resimulated experiments and baselines at this exact window. The naive method scales the observed window rate by clock time and ignores seasonal composition, so it is biased and over-confident away from a representative full-cycle window. The aware method reweights by the learned seasonal composition and propagates the extra uncertainty. See the full report.
